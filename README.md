# D-ployer-Keycloak-activer-le-MFA-et-configurer-OAuth2-OIDC
Projet Academic : Déploiement d’un serveur Keycloak, création d’un realm et d’un utilisateur, activation du MFA via TOTP et configuration d’un client OAuth2/OIDC utilisant le flux Authorization Code + PKCE pour tester une authentification moderne et sécurisée.
